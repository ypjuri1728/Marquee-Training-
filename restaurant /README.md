# 🍖 Restaurant Menu — HTML & CSS

A dark-themed restaurant menu card built using **pure HTML and CSS only** — no JavaScript, no frameworks.

---

## 📸 Preview

> Dark wooden board style menu with blurred background, gold text, circular food images and dot dividers.

---

## 📁 Project Structure
---

## ✨ Features

- Dark wooden board design
- Blurred background overlay using `backdrop-filter`
- Gold royal color theme
- Circular food images with gold border
- Dot dividers between sections
- `cursor: pointer` on all clickable items
- Hover effects — color change on text and image border
- 4 menu sections — Appetizer, Barbecue, Sides, Meat Skewers
- Food images loaded from Unsplash

---

## 🎨 CSS Concepts Used

| Concept | Where Used |
|---|---|
| `background-image` | Background food photo |
| `backdrop-filter: blur()` | Blur effect on overlay |
| `rgba()` | Dark transparent overlay |
| `border-radius: 50%` | Circular food images |
| `display: flex` | Row layout for each section |
| `position: fixed` | Overlay stays behind board |
| `:hover` | Color change on mouse over |
| `cursor: pointer` | Hand cursor on hover |

---

## 🖱️ Hover Effects

When mouse goes over a menu section:
- Section title color changes
- Item name turns white
- Image border color changes

No animation used — only color change  CSS `:hover`
